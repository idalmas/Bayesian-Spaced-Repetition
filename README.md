# Bayesian-Spaced-Repetition-

CS109 project 

Flask backend and Next frontent

## Getting Started

### Backend (Flask API)

1. **Install Python dependencies:**

    ```bash
    cd backend
    pip install -r requirements.txt
    ```

2. **Run the Flask app:**

    ```bash
    python app.py
    ```

   By default, the API should run on http://localhost:5000.

---

### Frontend (Next.js React)

1. **Install Node dependencies:**

    ```bash
    cd frontend
    npm install
    ```

2. **Start the development server:**

    ```bash
    npm run dev
    ```

   This will start the frontend, usually on http://localhost:3000.

---

Make sure both servers are running for full functionality.
